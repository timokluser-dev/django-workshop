# Django template

## Add Hosts file

Add the following to your `/etc/hosts` file.

    127.0.0.1       django.what-ever.lo

## Setup environment variables 

    make init

## Docker shortcuts

    make up
    make down
    make bash

## Django Admin GUI
You can access the Admin Gui through [http://django.what-ever.lo/admin/](http://django.what-ever.lo/admin/).
    
    Username: admin
    Password: admin
